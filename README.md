# ITSM-NG Docker CI Image

## Settings

The next secret configurations are required :

* DOCKER_HUB_USERNAME
* DOCKER_HUB_TOKEN
* GHCR_USERNAME
* GHCR_ACCESS_TOKEN

The next variable configurations are required :

* REPO_NAME
* USER_NAME

To set the configuration, go to `Settings > Secrets and variables > Actions`.
